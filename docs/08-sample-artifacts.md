### Section B: Short Answer (20 points, 5 questions)

**Q9.** What is the output of this code? (4 points)
```cpp
int x = 10;
int *p = &x;
*p = 20;
cout << x;
```

**Answer:** `20`

*Explanation: `p` points to `x`. `*p = 20` modifies the value at that address, which is `x`.*

---

**Q10.** Complete the blank to make this a valid string declaration: (4 points)
```cpp

```

**Answer:** `string` or `const char*`

---

**Q11.** What is the difference between `++i` and `i++`? (4 points)

**Answer:**
- `++i` (pre-increment): Increments first, then returns the new value
- `i++` (post-increment): Returns the current value, then increments

### Section C: Coding Tasks (40 points, 2 questions)

**Q14.** Write a function that takes an array of integers and its size, and returns the sum of all even numbers in the array. (20 points)

**Starter Code:**
```cpp
int sumOfEvens(int arr[], int size) {
    // Your code here
}
```

**Sample Solution:**
```cpp
int sumOfEvens(int arr[], int size) {
    int sum = 0;
    for (int i = 0; i < size; i++) {
        if (arr[i] % 2 == 0) {
            sum += arr[i];
        }
    }
    return sum;
}
```

**Q15.** Create a `Rectangle` class with the following specifications: (20 points)
- Private members: `length` and `width` (both `double`)
- Constructor that initializes both dimensions
- Member function `area()` that returns the area
- Member function `perimeter()` that returns the perimeter
- Member function `isSquare()` that returns `true` if length equals width

**Sample Solution:**
```cpp
class Rectangle {
private:
    double length;
    double width;

public:
    Rectangle(double l, double w) : length(l), width(w) {}

    double area() {
        return length * width;
    }

    double perimeter() {
        return 2 * (length + width);
    }

    bool isSquare() {
        return length == width;
    }
};
```

### Project: Student Grade Management System

#### Overview
Build a console-based application to manage student records and grades. This project integrates all concepts learned in the C++ Foundations course.

#### Requirements

##### Functional Requirements

1. **Student Management**
   - Add new student (name, ID, up to 5 grades)
   - View all students
   - Search student by ID
   - Delete student by ID

2. **Grade Calculations**
   - Calculate average grade per student
   - Determine letter grade (A: 90+, B: 80-89, C: 70-79, D: 60-69, F: <60)
   - Calculate class average

3. **Sorting & Display**
   - Sort students by ID
   - Sort students by average grade
   - Display formatted table output

4. **Persistence (Bonus)**
   - Save records to file
   - Load records from file

##### Technical Requirements

- Use a `Student` class with appropriate members
- Use arrays or vectors for storage
- Implement menu-driven interface
- Input validation
