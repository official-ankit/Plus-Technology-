## Design Philosophy

> **Google-like minimalism**: Every pixel earns its place. Speed is a feature. Clarity over cleverness.

### Core Principles
1. **Search-First**: Prominent search bar on every page
2. **Minimal Chrome**: No unnecessary UI elements
3. **High Contrast**: Clear visual hierarchy
4. **Ample Whitespace**: Room to breathe
5. **Keyboard-First**: All actions accessible via keyboard
6. **Mobile-First**: Designed for mobile, enhanced for desktop

## Rationale Summary

| Decision | Why |
|----------|-----|
| Inter font | Clean, highly legible, free, variable weights |
| 4px spacing base | Consistent rhythm, easy mental math |
| Focus rings | Critical for keyboard users, accessibility |
| Search-first | Fastest path to discovery (Google paradigm) |
| Bottom nav on mobile | Thumb-friendly, industry standard |
| Skeleton loaders | Perceived performance, reduces layout shift |
| Reduced motion | Respect user preferences, accessibility |
